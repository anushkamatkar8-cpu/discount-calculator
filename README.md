# Discount Calculator (Python)

## Description

This is a simple Python command-line application that allows users to enter a product price and discount percentage, then calculates the final discounted price.

## Features

* Accepts user input for price and discount percentage
* Calculates discount amount automatically
* Displays the final payable price
* Demonstrates use of Python input handling, arithmetic operations, and formatted output

## Technologies Used

* Python

## How to Run

1. Download the `.py` file
2. Run the program using:

```
python discount_calculator.py
```
